# CK3AGOT-ES-Docs
# AGOT ES - Documentación Oficial

Este repositorio contiene el estándar de traducción, glosarios y criterios de localización del mod A Game of Thrones en Español.

---

## Objetivo

Establecer un conjunto único de reglas de traducción y localización para asegurar coherencia en todo el proyecto.

---

## Secciones

- Criterios de traducción
- Reglas de sintaxis
- Reglas de identificación
- Glosario geográfico
- Glosario antropológico
- Glosario natural
- Glosario misceláneo

---

## Importante

Todo contribuidor del proyecto debe seguir estrictamente estas reglas. Cualquier excepción debe ser justificada y aprobada por el equipo de revisión.
