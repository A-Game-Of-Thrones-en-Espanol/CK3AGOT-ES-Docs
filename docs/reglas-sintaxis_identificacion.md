# TÉRMINOS Y REGLAS

## 1. REGLAS DE SINTAXIS

- **Palabras compuestas:** Se traducirán como una sola palabra.  
  (ej: Forkvalley → Valdeforca)

- **Palabras separadas por guion:** Se mantendrán como dos palabras separadas.  
  (ej: Fork_valley → Valle Forca)

- **Excepciones:**  
  Topónimos patronímicos, adaptaciones fonéticas, nombres canónicos y semicanónicos ignoran estas reglas.

---

## 2. REGLAS DE IDENTIFICACIÓN

Las palabras ya revisadas se marcan con las siguientes etiquetas:

- #CORREGIDO  
- #CORREGIDO-PATRONIMICO  
- #CORREGIDO-CANONICO  
- #CORREGIDO-ADAPTADO
