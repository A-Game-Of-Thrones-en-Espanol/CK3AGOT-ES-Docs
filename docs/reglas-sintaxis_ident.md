REGLAS DE SINTAXIS
Palabras compuestas: Se traducirán como una sola palabra (ej: Forkvalley -> Valdeforca).
Palabras separadas por guion: Se mantendrán como dos palabras separadas (ej: Fork_valley -> Valle Forca).
Topónimos patronímicos, adaptaciones fonéticas y nombres canónicos y semicanónicos ignoran estas reglas.
REGLAS DE IDENTIFICACIÓN
Las palabras ya revisadas se marcan con las siguientes etiquetas:
CORREGIDO
CORREGIDO-PATRONIMICO
CORREGIDO-CANONICO
CORREGIDO-ADAPTADO
